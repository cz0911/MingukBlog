关于网站

  • 本网站是一个个人博客网站，主要分享博主的编程学习心得
  • 网站主要使用 Django2.0 + Bootstrap4 搭建，源码在博主 Github 中， 目前部署在阿里云轻量应用服务器

功能介绍

  • Django 自带的后台管理系统，方便对于文章、用户及其他动态内容的管理
  • 文章分类、浏览量统计
  • 用户登录注册系统
  • 文章评论系统，二级评论结构和回复功能
  • django-redis 支持的缓存系统，遵循缓存原则，加速网站打开速度

网站支持

  • 前端使用 Bootstrap4 + jQuery 支持响应式；图标使用 Font Awesome
  • 后端 Python 3.6，Django 2.0，其他依赖查看源码中 requirements.txt
  • 后台数据库使用 MySQL，缓存数据库使用 Redis
  • 网站部署使用 Nginx + gunicorn，之前使用的 Nginx + uwsgi
  • django-jet 用于美化后台管理系统，变成响应式界面
  • django-ckeditor 等用于第三方评论库
  • django-redis 缓存
